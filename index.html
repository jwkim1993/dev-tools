DevTools

    Developer Tools

    Free, fast, and private developer utilities — all processing happens in your browser

    JSON Formatter

    Format, validate, minify, and explore JSON data with tree view and statistics. Supports drag-and-drop, URL loading, and real-time error detection.

    UUID Generator

    Generate UUID v1, v4, and v7 identifiers with bulk generation, format options, and one-click copy. RFC 9562 compliant.

    JWT Decoder

    Decode, inspect, verify, and generate JSON Web Tokens. Claims inspector with expiry detection, HMAC signature verification, and token builder.

    Cron Expression

    Parse cron expressions into plain English, build schedules visually, and preview next execution times. Supports standard 5-field cron syntax with presets.

    Regex Tester

    Write, test, and debug regular expressions with real-time match highlighting, capture group inspection, and replace preview. All JavaScript flags supported.

    Encoder / Decoder

    Encode and decode Base64, URL (percent-encoding), and HTML entities. Real-time conversion with swap, copy, and character count.

    Diff Viewer

    Compare two texts side-by-side or in unified view with line-level and character-level diff highlighting. Copy diff output for code reviews.
